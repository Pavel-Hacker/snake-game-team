# snake-game-team
Веб игра Snake - коллективная разработка
# Команда разработки

- **Павел** - Тимлид, инфраструктура, HTML
- **Сергей** - UI/Дизайн, CSS, адаптивная верстка  
- **Алексей** - Игровая логика, JavaScript, механики

# Как играть

- **Стрелки** или **WASD** - управление змейкой
- **R** - рестарт игры
- **Цель** - собирать еду и избегать столкновений

# Запуск

1. Скачайте файлы проекта
2. Откройте index.html в браузере
3. Нажмите "Старт" для начала игры

# Структура проекта

- index.html - основная структура
- style.css - стили и дизайн
- game.js - игровая логика
